## Tech Stack

- Next.js 14 (App Router)
- Tailwind CSS
- Sora + Outfit fonts
- Scroll-triggered animations
- Deployed on Vercel

## Getting Started

```bash
git clone https://github.com/Yerima18/portfolio.git
cd portfolio
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)
